# nlp-insight-extraction-reviews
NLP-based insight extraction from unstructured customer reviews using BERTopic and Sentence Transformers
## 📌 Project Overview
This project focuses on extracting meaningful insights from unstructured customer reviews using Natural Language Processing (NLP).

Instead of traditional sentiment analysis, this project applies topic modeling (BERTopic) to uncover hidden patterns and customer concerns.

---

## 📊 Dataset
- Source: Yelp Review Dataset (Hugging Face)
- Type: Unstructured text data
- Sample Size: 20,000 reviews

---

## ⚙️ Technologies Used
- Python
- Pandas
- Sentence Transformers
- BERTopic
- Scikit-learn

---

## 🔄 Project Workflow
1. Data Loading  
2. Exploratory Data Analysis (EDA)  
3. Data Cleaning  
4. Text Embedding using Sentence Transformers  
5. Topic Modeling using BERTopic  
6. Insight Extraction  
7. Visualization  

---

## 🧠 Key Insights

- Food and restaurant-related reviews are the most dominant  
- Customers frequently discuss service quality and staff behavior  
- Automotive services show complaints about delays and communication  
- Beauty and wellness services are commonly reviewed  
- Travel-related issues include flight delays and airport experience  

---

## 📈 Results

- Extracted meaningful topics from customer reviews  
- Identified major areas of customer concern  
- Converted unstructured data into actionable insights
